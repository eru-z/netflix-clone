# Netflix Clone (PHP & MySQL)

A Netflix-style streaming platform clone built using core PHP and MySQL.

## Features
- User registration & login
- Session-based authentication
- Movies & TV shows browsing
- My List functionality
- Admin dashboard
- Dynamic content loaded from database

## Tech Stack
- PHP (core PHP)
- MySQL
- HTML, CSS, JavaScript

## Purpose
This project was built to practice backend development, authentication, and database-driven web applications.
